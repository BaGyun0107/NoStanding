## **ABOUT**


**예약 서비스는 많은 실태**

**종합으로 되어있는 예약 서비스가 잘 없는 현실**

**여러 방면의 가게를 서비스 하나로 모두 예약 할 수 있는 종합 예약 서비스**

**NoStanding을 통해 사용 가능한 기능**

* 카테고리 선택을 통한 필터링 / 검색
* 시간대 별 예약 시스템
* 리뷰 시스템으로 가게의 솔직한 평판 확인 가능
* 예약 한 것을 미리 알려주는 단계 별 알람 시스템
* 소셜 로그인을 통한 가입 절차 간편화
* 점주들에게는 다른 UI를 제공하여 예약 시스템 관리 편의성 개선
